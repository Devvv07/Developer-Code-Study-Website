BCA Python Interview Questions

Basic Interview Questions

Q1. What is Python and why is it popular? Easy

Answer: Python is a high-level, interpreted programming language known for its simplicity and readability. It's popular due to easy syntax, vast libraries (NumPy, Pandas, Django), versatility (web, AI, data science), and strong community support.

Q2. Explain the difference between append() and extend() methods. Easy

Answer: append() adds a single element to the end of list, while extend() adds multiple elements from an iterable. Example: list.append([1,2]) adds [1,2] as one element, list.extend([1,2]) adds 1 and 2 separately.

Q3. What is the difference between // and / operators? Easy

Answer: / is true division (returns float), while // is floor division (returns integer by rounding down). Example: 7/2 = 3.5, but 7//2 = 3

Q4. What is the difference between is and ==? Easy

Answer: == compares values/content of objects, while 'is' compares object identity (memory address). Example: [1,2] == [1,2] is True, but [1,2] is [1,2] is False (different objects).

Q5. What are mutable and immutable data types in Python? Easy

Answer: Mutable: Can be changed after creation (list, dict, set). Immutable: Cannot be changed (str, tuple, int). Example: list.append(1) modifies the list, but str.upper() creates a new string.

Intermediate Questions

Q6. Explain *args and **kwargs. Medium

Answer: *args allows passing variable number of non-keyword arguments as tuple. **kwargs allows passing variable number of keyword arguments as dictionary. Used when exact number of arguments is unknown.

Q7. What is the difference between shallow copy and deep copy? Medium

Answer: Shallow copy creates new object but references to nested objects are same. Deep copy creates completely independent copy including nested objects. Use copy.copy() for shallow and copy.deepcopy() for deep copy.

Q8. Explain list comprehension vs generator expression. Medium

Answer: List comprehension [x for x in range(10)] creates entire list in memory. Generator expression (x for x in range(10)) creates iterator, yielding values on-demand. Generators are memory efficient for large datasets.

Q9. What are Python decorators and how do they work? Medium

Answer: Decorators are functions that modify behavior of other functions. They use @decorator syntax. Commonly used for logging, authentication, caching. Example: @staticmethod, @property, @classmethod

Q10. Explain how exception handling works in Python. Medium

Answer: Use try-except-else-finally blocks. Try contains risky code, except catches specific exceptions, else runs if no exception, finally always executes. Can raise exceptions with raise keyword and create custom exceptions.

Advanced Questions

Q11. What is the Global Interpreter Lock (GIL)? Hard

Answer: GIL is a mutex that allows only one thread to execute Python bytecode at a time. It prevents race conditions but limits multi-threading performance for CPU-bound tasks. Use multiprocessing for parallel CPU operations.

Q12. Explain Python's memory management. Hard

Answer: Python uses automatic memory management with reference counting and garbage collection. Objects are allocated on heap. When reference count reaches zero, memory is freed. Cyclic garbage collector handles circular references.

Q13. Explain the difference between __init__ and __new__. Hard

Answer: __new__ is called first to create instance (class method), __init__ initializes the created instance (instance method). __new__ returns new instance, __init__ returns None. __new__ rarely overridden except for immutable types.

Q14. What is metaclass in Python? Hard

Answer: Metaclass is a class of a class. It defines how a class behaves. Classes are instances of metaclasses. Used for advanced customization like registering classes or adding methods dynamically.

Q15. Explain Python's context managers. Medium

Answer: Context managers handle resource management using with statement. Implement __enter__ and __exit__ methods. Example: file handling with open() as f. Ensures cleanup even if exceptions occur.

Coding Interview Questions

Q16. Write a function to check if a string is palindrome. Medium

Answer:

def is_palindrome(s):
    return s == s[::-1]

print(is_palindrome("radar"))  # True

Q17. Implement Fibonacci series using recursion. Medium

Answer:

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(5))  # 5

Q18. Write a program to find duplicates in a list. Easy

Answer:

def find_duplicates(lst):
    seen = set()
    duplicates = set()
    for item in lst:
        if item in seen:
            duplicates.add(item)
        else:
            seen.add(item)
    return list(duplicates)

print(find_duplicates([1, 2, 2, 3, 3, 3]))  # [2, 3]

Q19. Implement a function to sort a list using bubble sort. Medium

Answer:

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

print(bubble_sort([64, 34, 25, 12, 22, 11, 90]))  # [11, 12, 22, 25, 34, 64, 90]
