BTech Java Paperset

Short Answer Questions (2-3 Marks)

Q1. What is the difference between JDK, JRE, and JVM? 3 Marks

Answer:

  • JDK (Java Development Kit): Complete development kit with compiler, debugger, and tools
  • JRE (Java Runtime Environment): Runtime environment to execute Java programs
  • JVM (Java Virtual Machine): Virtual machine that executes bytecode

Q2. Explain the concept of constructor in Java. 3 Marks

Answer: A constructor is a special method that is called when an object is instantiated. It has the same name as the class and no return type. Used to initialize object state.

Q3. What is method overloading and overriding? 3 Marks

Answer:

  • Overloading: Same method name with different parameters in the same class (compile-time polymorphism)
  • Overriding: Child class redefines parent class method with same signature (runtime polymorphism)

Q4. Difference between String and StringBuffer. 2 Marks

Answer: String is immutable (cannot be changed), while StringBuffer is mutable (can be modified). StringBuffer is thread-safe.

Q5. What is the main method in Java? 2 Marks

Answer: The main method is the entry point of a Java program. It has signature: public static void main(String[] args). JVM calls this method to start execution.

Q6. What is the difference between == and equals() method? 3 Marks

Answer: == compares object references (memory addresses), while equals() compares content/values. For strings, equals() compares character sequences.

Long Answer Questions (5-10 Marks)

Q7. Explain the concept of inheritance with an example. 5 Marks

Answer: Inheritance is a mechanism where a new class acquires properties and methods of an existing class. The existing class is called parent/super class and new class is called child/sub class.

Types: Single, Multilevel, Hierarchical, Multiple (through interfaces)

Q8. What is exception handling? Explain try-catch-finally block. 7 Marks

Answer: Exception handling is a mechanism to handle runtime errors. Try block contains code that might throw exception, catch block handles the exception, and finally block always executes regardless of exception.

Q9. Explain different types of access modifiers in Java. 5 Marks

Answer:

  • private: Accessible only within the class
  • default: Accessible within the package
  • protected: Accessible within package and subclasses
  • public: Accessible everywhere

Q10. What is multithreading? Explain thread lifecycle. 7 Marks

Answer: Multithreading is concurrent execution of multiple threads. Thread lifecycle states: New, Runnable, Running, Blocked/Waiting, Terminated.

Q11. Explain abstract class and interface with differences. 7 Marks

Answer: Abstract class can have abstract and concrete methods. Interface can only have abstract methods (before Java 8). A class can extend one abstract class but implement multiple interfaces.

Q12. What is Java Collections Framework? Explain List, Set, and Map. 10 Marks

Answer: Collections Framework provides architecture to store and manipulate groups of objects.

  • List: Ordered collection, allows duplicates (ArrayList, LinkedList)
  • Set: Unordered collection, no duplicates (HashSet, TreeSet)
  • Map: Key-value pairs, unique keys (HashMap, TreeMap)

Programming Questions

Q13. Write a Java program to check if a number is prime. 5 Marks

Answer:

public class PrimeCheck {
    public static boolean isPrime(int n) {
        if (n <= 1) return false;
        for (int i = 2; i <= Math.sqrt(n); i++) {
            if (n % i == 0) return false;
        }
        return true;
    }
    
    public static void main(String[] args) {
        System.out.println(isPrime(7)); // true
    }
}

Q14. Write a program to reverse a string in Java. 3 Marks

Answer:

public class ReverseString {
    public static void main(String[] args) {
        String str = "hello";
        String reversed = new StringBuilder(str).reverse().toString();
        System.out.println(reversed); // olleh
    }
}

Q15. Write a program to implement inheritance in Java. 5 Marks

Answer:

class Animal {
    void sound() {
        System.out.println("Animal makes sound");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Dog barks");
    }
    
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.sound(); // Dog barks
    }
}
