BSc Web Development Notes

1. Introduction to Web Development

Web Development is the process of creating websites and web applications.

  • Frontend: Client-side (HTML, CSS, JavaScript)
  • Backend: Server-side (Node.js, Python, PHP)
  • Database: Data storage (MySQL, MongoDB)
  • Full Stack: Frontend + Backend

2. HTML Fundamentals

HTML (HyperText Markup Language) is the standard markup language for web pages.

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Welcome</h1> <p>This is a paragraph.</p> <a href="page.html">Link</a> <img src="image.jpg" alt="Description"> </body> </html>

Common Tags:

  • <div>, <span> - Containers
  • <h1> to <h6> - Headings
  • <p>, <br> - Paragraph, Line break
  • <ul>, <ol>, <li> - Lists
  • <table>, <tr>, <td> - Tables
  • <form>, <input>, <button> - Forms

3. CSS Fundamentals

CSS (Cascading Style Sheets) is used to style HTML elements.

/* Selectors */ h1 { color: #e91e63; font-size: 32px; text-align: center; } .container { width: 80%; margin: 0 auto; padding: 20px; } #header { background: linear-gradient(135deg, #e91e63, #9c27b0); }

CSS Box Model: Content ‚Üí Padding ‚Üí Border ‚Üí Margin

Layout Methods: Flexbox, Grid, Float, Position

4. JavaScript Basics

JavaScript adds interactivity to web pages.

// Variables let name = "John"; const age = 25; // Functions function greet(name) { return `Hello, ${name}!`; } // DOM Manipulation document.getElementById('btn').addEventListener('click', () => { document.querySelector('.text').innerHTML = 'Changed!'; }); // Arrays const fruits = ['apple', 'banana', 'orange']; fruits.forEach(fruit => console.log(fruit));

5. Responsive Design

Creating websites that work on all devices.

/* Media Queries */ @media (max-width: 768px) { .container { width: 100%; padding: 10px; } } /* Flexible Layout */ .flex-container { display: flex; flex-wrap: wrap; justify-content: space-between; }
  • Mobile-first approach
  • Viewport meta tag
  • Flexible images
  • Media queries

6. Modern CSS (Flexbox & Grid)

Flexbox: One-dimensional layout

.flex-container { display: flex; justify-content: center; align-items: center; gap: 20px; }

Grid: Two-dimensional layout

.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; }

7. JavaScript ES6+ Features

// Arrow Functions const add = (a, b) => a + b; // Destructuring const { name, age } = person; const [first, second] = array; // Template Literals const message = `Hello, ${name}!`; // Promises & Async/Await async function fetchData() { const response = await fetch('api/data'); const data = await response.json(); return data; }

8. Web Development Best Practices

  • Semantic HTML: Use meaningful tags
  • Accessibility: ARIA labels, alt text
  • Performance: Minify CSS/JS, optimize images
  • SEO: Meta tags, structured data
  • Version Control: Git and GitHub
  • Cross-browser Testing: Test on multiple browsers
