BCA Python Important Questions

Short Answer Questions (2-3 Marks)

Q1. What are the key features of Python? 3 Marks

Answer:

  • Easy to learn and read
  • Interpreted language
  • Dynamically typed
  • Object-oriented
  • Large standard library
  • Cross-platform compatibility

Q2. Explain mutable and immutable data types in Python. 3 Marks

Answer:

  • Mutable: Can be changed after creation (List, Dictionary, Set)
  • Immutable: Cannot be changed after creation (String, Tuple, Integer, Float)

Q3. What is the difference between list and tuple? 2 Marks

Answer: List is mutable and uses square brackets [], while tuple is immutable and uses parentheses (). Lists are slower but flexible, tuples are faster but fixed.

Q4. What is a lambda function? 2 Marks

Answer: Lambda is an anonymous function defined using lambda keyword. It can take any number of arguments but has only one expression. Example: square = lambda x: x**2

Q5. What is PEP 8? 2 Marks

Answer: PEP 8 is the style guide for Python code. It provides conventions for writing readable and consistent code, including indentation, naming, and spacing.

Q6. Explain the use of 'self' in Python classes. 3 Marks

Answer: 'self' is a reference to the current instance of the class. It is used to access variables and methods of the class within the class definition.

Long Answer Questions (5-10 Marks)

Q7. Explain different types of operators in Python with examples. 7 Marks

Answer:

  • Arithmetic: +, -, *, /, %, **, //
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: and, or, not
  • Assignment: =, +=, -=, *=, /=
  • Membership: in, not in
  • Identity: is, is not

Q8. What are Python decorators? Explain with example. 5 Marks

Answer: Decorators are functions that modify the behavior of other functions. They use @decorator_name syntax above function definition. Example: @staticmethod, @property

Q9. Explain file handling in Python with modes. 7 Marks

Answer: File handling allows reading and writing files. Modes:

  • r: Read mode (default)
  • w: Write mode (overwrites)
  • a: Append mode
  • r+: Read and write
  • rb, wb: Binary modes

Q10. What is list comprehension? Explain with examples. 5 Marks

Answer: List comprehension provides concise way to create lists. Syntax: [expression for item in iterable if condition]. Example: squares = [x**2 for x in range(10)]

Q11. Explain inheritance in Python with types. 7 Marks

Answer: Inheritance allows a class to inherit attributes and methods from another class.

  • Single Inheritance: One parent, one child
  • Multiple Inheritance: Multiple parents, one child
  • Multilevel Inheritance: Chain of inheritance
  • Hierarchical Inheritance: One parent, multiple children

Q12. What are modules and packages in Python? 10 Marks

Answer:

  • Module: A file containing Python code (.py file). Can be imported using import statement
  • Package: A directory containing multiple modules with __init__.py file
  • Built-in modules: math, random, datetime, os, sys
  • Third-party packages: numpy, pandas, requests

Programming Questions

Q13. Write a Python program to check if a number is prime. 5 Marks

Answer:

def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

print(is_prime(7))  # True

Q14. Write a program to reverse a string. 3 Marks

Answer:

def reverse_string(s):
    return s[::-1]

print(reverse_string("hello"))  # olleh

Q15. Write a program to find factorial using recursion. 5 Marks

Answer:

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

print(factorial(5))  # 120
