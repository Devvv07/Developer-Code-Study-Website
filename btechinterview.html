Q1. What are the main features of Java? Easy

Answer: Platform independent, Object-oriented, Simple, Secure, Robust, Multithreaded, Architecture-neutral, Portable, High performance, Distributed, Dynamic.

Q2. Explain garbage collection in Java. Medium

Answer: Automatic memory management by JVM. Removes unreferenced objects using Mark & Sweep, Generational GC.

Q3. Difference between == and equals()? Easy

Answer: == checks reference, equals() checks content. Use equals() for String comparison.

Q4. HashMap internal working? Hard

Answer: Array of buckets ‚Üí hashCode() ‚Üí bucket ‚Üí equals() ‚Üí linked list/tree (Java 8+). Rehash at 0.75 load factor.

Q5. final vs finally vs finalize? Medium

  • final: Constant, no override/inheritance
  • finally: Always executes after try-catch
  • finalize: Called by GC before object destruction
