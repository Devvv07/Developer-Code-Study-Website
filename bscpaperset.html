BSc Web Development Paper Set

Short Answer Questions (2-3 Marks)

Q1. What is the difference between HTML and HTML5? 3 Marks

Answer: HTML5 introduced new semantic elements (header, footer, article, section), multimedia support (audio, video), canvas for graphics, local storage, geolocation API, and improved form controls.

Q2. Explain the CSS Box Model. 3 Marks

Answer: CSS Box Model consists of Content, Padding (space around content), Border (surrounds padding), and Margin (space outside border). Total width = content + padding + border + margin.

Q3. What is the DOM in JavaScript? 2 Marks

Answer: DOM (Document Object Model) is a programming interface for HTML documents. It represents the page structure as a tree of objects, allowing JavaScript to manipulate HTML elements dynamically.

Q4. Difference between inline, block, and inline-block elements. 3 Marks

Answer: Inline elements don't start new line, can't set width/height. Block elements start new line, take full width. Inline-block combines both: doesn't break line but accepts width/height.

Q5. What is the viewport meta tag? 2 Marks

Answer: Viewport meta tag controls page scaling and layout on mobile devices. Example: <meta name="viewport" content="width=device-width, initial-scale=1.0"> ensures proper rendering on different screen sizes.

Long Answer Questions (5-10 Marks)

Q6. Explain Flexbox layout with properties. 7 Marks

Answer: Flexbox is one-dimensional layout for arranging items.

  • Container properties: display: flex, flex-direction, justify-content, align-items, flex-wrap, gap
  • Item properties: flex-grow, flex-shrink, flex-basis, align-self, order
  • Best for navigation bars, cards layout, centering elements

Q7. What is responsive web design? Explain techniques. 7 Marks

Answer: Responsive design makes websites adapt to different screen sizes.

  • Fluid layouts: Use percentages instead of pixels
  • Media queries: Apply styles based on screen size
  • Flexible images: max-width: 100%
  • Viewport meta tag: Control page scaling
  • Mobile-first approach: Design for mobile, then desktop

Q8. Explain event handling in JavaScript. 5 Marks

Answer: Events are actions that happen in browser (click, hover, submit). Event listeners respond to these actions using addEventListener(). Event object contains info about the event. Event bubbling and capturing control event propagation.

Q9. What are semantic HTML tags? Give examples. 5 Marks

Answer: Semantic tags clearly describe their meaning to browser and developer.

  • <header> - Page/section header
  • <nav> - Navigation links
  • <main> - Main content
  • <article> - Independent content
  • <section> - Thematic grouping
  • <footer> - Page/section footer
  • Benefits: Better SEO, accessibility, code readability

Q10. Explain CSS Grid layout system. 7 Marks

Answer: CSS Grid is two-dimensional layout system for rows and columns.

  • Properties: display: grid, grid-template-columns, grid-template-rows, gap, grid-column, grid-row
  • Features: Explicit placement, responsive without media queries, overlapping elements
  • Use cases: Page layouts, complex designs, dashboards

Q11. What are JavaScript promises and async/await? 10 Marks

Answer:

  • Promises: Objects representing eventual completion/failure of async operation. Three states: pending, fulfilled, rejected. Uses .then() and .catch()
  • Async/Await: Syntactic sugar over promises. Makes async code look synchronous. async function returns promise. await pauses execution until promise resolves
  • Benefits: Better error handling, cleaner code, easier to read than callbacks

Programming Questions

Q12. Write JavaScript code to validate an email address. 5 Marks

Answer:

function validateEmail(email) {
    const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return regex.test(email);
}

console.log(validateEmail("test@example.com")); // true

Q13. Write CSS for a responsive navigation bar. 7 Marks

Answer:

.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
    background: #333;
}

.nav-links {
    display: flex;
    list-style: none;
}

.nav-links li {
    margin: 0 1rem;
}

.nav-links a {
    color: white;
    text-decoration: none;
}

@media (max-width: 768px) {
    .nav-links {
        display: none;
    }
}

Q14. Write HTML for a contact form with validation. 5 Marks

Answer:

    <input type="text" id="name" placeholder="Name" required>
    <input type="email" id="email" placeholder="Email" required>
    <textarea id="message" placeholder="Message" required></textarea>
